# Assets

Drop production icons here before running `npm run build`:

- `icon.png` (1024×1024) — universal
- `icon.ico` — Windows
- `icon.icns` — macOS
- `tray-icon.png` (22×22)
- `tray-icon@2x.png` (44×44)
- `tray-icon-active.png` — used while on an active call
- `installer-header.bmp` — NSIS installer header (optional)
- `dmg-background.png` — DMG background (optional)
- `license.txt` — shown in the NSIS installer
